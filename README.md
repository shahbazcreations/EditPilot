# EditPilot

EditPilot is an Adobe Premiere Pro extension built to automate repetitive editing tasks for faceless YouTube editors.

## Goal

Reduce editing time by automating:

- Sequential media replacement
- Auto importing images
- Layout templates
- Auto fitting images
- Scene building
- AI assisted asset collection (future)
